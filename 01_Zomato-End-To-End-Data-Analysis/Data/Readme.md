# Raw Data — Zomato Sales & User Performance Analysis

This folder contains the original Zomato restaurant dataset used for this project.

---

## Files in This Folder

| File Name | Description |
|-----------|-------------|
| Zomato_Dataset.xlsx | Dataset for analysis |

---

## Dataset Details

| Property | Value |
|----------|-------|
| **Source** | Zomato Restaurant Dataset |
| **Tables** | Orders, Restaurants, Users, Menu, Food |
| **Cities** | Multiple Cities |
| **Restaurants** | Multiple Restaurants |
| **Key Metrics** | Sales, Orders, Users, Ratings, Cities, Restaurants|

---

## Key Fields

| Field | Description |
|-------|-------------|
| **`Order ID`** | Unique identifier for each order |
| **`User ID`** | Unique identifier for each customer |
| **`Restaurant Name`** | Name of the restaurant |
| **`City`** | City where the order was placed |
| **`Cuisine`** | Type of cuisine served |
| **`Order Date`** | Date of the order |
| **`Item Count`** | Number of items ordered |
| **`Total Amount`** | Total order value |
| **`Rating`** | Customer rating for the restaurant |

---

## Data Cleaning (Power Query & Excel)

- Removed duplicate records.
- Handled missing and null values.
- Corrected data types for dates and numeric fields.
- Built relationships between tables for dashboard analysis.

---

## Purpose

The dataset was cleaned and transformed before importing into Power BI to create interactive dashboards for:

- Sales Performance Analysis
- User Behavior Analysis
- City-wise Performance
- Restaurant Performance
- Cuisine Trend Analysis
- Business Insights & KPI Reporting
